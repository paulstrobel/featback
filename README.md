# f-eat-back

F-eat-back ist ein Web-basiertes Bewertungstool für das Kantinenessen in Karlsruhes Hochschulen. Entwicklet und getestet wird das Projekt als Pilotprojekt an der Dualen Hochschule Karlsruhe.
